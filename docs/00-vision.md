# Vision

We are building a mobile-first Tower Defense game inspired by classic Warcraft 3 custom TD maps.

The player builds towers on a grid to create a maze-like path for creeps. Towers act as obstacles, but the player must never be allowed to fully block the path from entrance to exit.

The first target platform is browser-based mobile gameplay via PWA or Telegram Web App. The game should later support multiplayer and more advanced online systems, but the MVP is single-player only.

## Core fantasy

- Build a clever maze.
- Survive waves of creeps.
- Upgrade towers.
- Optimize economy.
- Feel the nostalgic Warcraft 3 custom map vibe in pixel-art style.

## Long-term ideas

- Multiplayer via WebRTC.
- Dynamic maps.
- Multiple creep factions inspired by undead, orcs, humans, elves.
- Tower modifiers: poison, frost, splash, armor break.
- Progression, records, and online features.
